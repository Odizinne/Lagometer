# Lagometer
PyQt6 ping app.

![image](https://github.com/Odizinne/Lagometer/assets/102679854/b377d6eb-645a-473d-9a5c-fcb3f36d6049)

## Requirements

### Runtime

- PyQt6
- ping3

`pip install pyqt6 ping3`

Then run: `python lagometer.py`

### Build dependencies

- cx_freeze
- Everything listed above

`pip install pyqt6 ping3 cx_freeze`

Then run: `python setup.py build`
