# Lagometer
PyQt6 ping app.

![image](https://github.com/Odizinne/Lagometer/assets/102679854/9fbeefdb-661a-4583-a9be-4b4153c2dc24)

## Requirements

### Runtime

- PyQt6
- ping3

`pip install pyqt6 ping3`

Then run: `python lagometer.py`

### Build dependencies

- cx_freeze
- Everything listed above

`pip install pyqt6 ping3 cx_freeze`

Then run: `python setup.py build`
